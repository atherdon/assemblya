# AssemBLYA

Veterans-led engineering project building a **modular payload delivery platform** that is simpler, more reliable, and more affordable with every generation.

:::tip Start here
- [Why we build this](./mission.md)
- [Українською](./ua.md)

:::

## At a glance

| | |
| --- | --- |
| **Goal** | One modular platform for many payload missions |
| **Model** | Bulk supply to units and partners — not retail |
| **Approach** | Reinvest into production, testing, and R&D instead of maximizing margin |

## Principles

Every new version should be:

- simpler
- more reliable
- easier to manufacture and repair
- significantly more affordable
