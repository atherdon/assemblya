# AssemBLYA

Ветеранський інженерний проєкт, що будує **модульну платформу скиду корисного навантаження**, яка з кожним поколінням стає простішою, надійнішою та доступнішою.

:::tip Почніть тут
- [Чому ми це робимо](./mission.md)
- [Дорожня карта та лінійка продуктів](../specifications/roadmap.md)
:::

## Коротко

| | |
| --- | --- |
| **Мета** | Одна модульна платформа для багатьох місій |
| **Модель** | Гуртові поставки підрозділам і партнерам — не роздріб |
| **Підхід** | Реінвестиції у виробництво, тестування та R&D замість максимізації маржі |

## Принципи

Кожне нове покоління має бути:

- простішим
- надійнішим
- легшим у виробництві та ремонті
- значно доступнішим за ціною
